# Справочные таблицы

## Статус

Заморожено (механически завершено)

## Назначение

Этот раздел содержит полные справочные таблицы, извлечённые из исходного кода DSNAtlantis.

Источник таблиц:

- `DSNAtlantis-src/standard/rules.h`
- `DSNAtlantis-src/standard/rules.cpp`
- `DSNAtlantis-src/standard/spells.cpp`
- `DSNAtlantis-src/standard/skillshows.cpp`

Документы:

- [skills.md](skills.md) — полный список `SkillDefs`.
- [items.md](items.md) — полный список `ItemDefs`.
- [spells.md](spells.md) — все магические навыки и заклинания.

## Правило чтения

Таблицы фиксируют данные исходного сервера. Если старый текст правил расходится с таблицами или поведением кода, каноническим считается код.

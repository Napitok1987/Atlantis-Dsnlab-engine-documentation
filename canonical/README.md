# Каноническая документация

Этот каталог содержит каноническую документацию по Atlantis DSNlab.

Исторический код оригинального сервера используется для проверки как исходный репозиторий:

- [Labutin/DSNAtlantis](https://github.com/Labutin/DSNAtlantis)

Правила для документов `as-is`:

- документы `as-is` не требуют внешних источников для чтения;
- документы не содержат проектных решений для новой игры;
- документы описывают только поведение оригинала;
- спорные места формулируются как часть канона только после проверки в исследовательском слое;
- если источники расходятся, в канон попадает поведение, подтверждённое кодом.

Структура:

- `as-is/` — каноническое описание оригинальной игры;
- `reference/` — полные справочные таблицы из исходного кода.

Текущий статус:

- каркас каталога создан;
- базовый комплект `as-is/` уже заполнен по основным системам;
- все документы `as-is/` переведены в статус `Заморожено (механически завершено)`;
- незакрытых расхождений высокого и среднего приоритета сейчас не зафиксировано;
- для `as-is` уже существует набор исполняемых регрессионных тестов игровых механик;
- дальнейшая работа должна идти вне замороженного `as-is`: публикация и следующий нормализованный слой.
